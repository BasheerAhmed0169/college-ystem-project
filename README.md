#  Enterprise College Network System Design

An enterprise-grade, secure, and scalable campus network architecture designed and simulated using **Cisco Packet Tracer**. 

---

##  Key Features
* **Traffic Isolation:** VLAN segmentation for Students, Faculty, Admin, and Guests.
* **Auto IP Assignment:** Centralized DHCP configuration for dynamic IP routing.
* **Secure Internet Gateway:** NAT/PAT implementation for public-private IP mapping.
* **Network Security:** SSH remote access and switch port security to prevent unauthorized access.
* **Routing Infrastructure:** Efficient Inter-VLAN routing for cross-department communication.

---

##  Tech Stack & Tools
* **Software:** Cisco Packet Tracer (v8.0+)
* **Topology:** Core, Distribution, and Access layers
* **Protocols:** DHCP, NAT, Inter-VLAN Routing, SSH, Port-Security

---

##  How to Use
1. Download and open **`College Network.pkt`** in Cisco Packet Tracer.
2. Let the link lights turn green.
3. Test end-to-end connectivity using the ping command or packet PDU tool!
